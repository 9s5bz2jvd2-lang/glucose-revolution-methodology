## 三、知识体系架构

### 三大模块

1. **基础科学模块**（TKP001-044）
   - 葡萄糖是什么、从哪里来、如何进入血液
   - 血糖峰值的定义、检测、分析
   - 覆盖全书第一部分

2. **危害机制模块**（TKP045-060）
   - 三大危害：氧化应激、糖化反应、胰岛素过量
   - 10个短期影响 + 11个长期危害
   - 覆盖全书第二部分

3. **实践策略模块**（TKP061-080）
   - 10个控糖窍门（原理+实践）
   - 组合策略、场景应用、一日示范
   - 覆盖全书第三部分+附录

### 核心创新

1. **果糖峰值"冰山模型"**：血糖仪只能看到水面上（葡萄糖），水下的果糖峰值更危险
2. **三大机制因果图**：氧化应激→糖化→炎症的恶性循环
3. **窍门组合策略**：叠加效应，4个窍门同时使用可降低60-75%峰值
4. **三路径学习设计**：基础4周/临床2周/实践1周

## 四、使用指南

### 对于学习者
1. 从TKP001开始，按编号顺序学习
2. 或选择学习路径（基础/临床/实践）
3. 每学完一个TKP，完成基础活动
4. 定期自测（每章末的评估标准）

### 对于教师
1. 使用"教学法建议"字段设计课堂
2. 结合学习活动设计小组讨论
3. 利用评估标准设计考试/测验
4. 补充"相关资源"中的文献

### 对于开发者
1. TKP可直接转化为在线课程模块
2. 知识图谱可转化为互动式学习路径
3. 评估标准可转化为自动测验系统
4. 学习活动可转化为实践作业
